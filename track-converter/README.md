# Line Rider Format Converter, in Rust

> [!WARNING]
> This project is in an unfinished state. Use at your own risk.

A CLI for converting between Line Rider track formats, written in rust.

## Quickstart

```bash
# Build from source (releases not supported yet)
make build-cli
# MacOS/Linux
target/releases/track-converter ./samples/HAM.trk json
```
